## ✨ 新增功能

- **WSL 自动映射开关** — 每个设备可独立开启自动映射，插上设备后自动 attach 到 WSL，无需手动操作
- **映射加载动画** — 操作中端口列显示 spinner + 方向指示（如 `COM3 → WSL`），直观展示映射进度

## 🔧 改进

- **启动性能优化** — SetupAPI 从逐端口查询改为一次性建立映射表，复杂度 O(N×M) → O(N+M)
- **异步 HTTP 请求** — 更新检查和下载改用异步 reqwest，不再阻塞线程池
- **端口列表缓存** — 多监视器启动时共享 2 秒内的查询结果，避免重复调用
- **延迟更新检查** — 首次启动延迟 2 秒检查更新，优先完成 UI 渲染
- **WSL 设备列表按 BUSID 排序** — 刷新不再改变排序顺序
- **WSL 映射并发执行** — attach/detach 改用 `async + spawn_blocking`，多个端口可同时映射
- **列表合并刷新** — 取消映射后设备不再消失，后台刷新保留现有设备
- **并发防抖** — 操作中的设备自动禁用复选框，防止重复点击
- **更新检查双通道** — 先直连 GitHub，失败再走镜像，解决镜像不可用时无法更新的问题
- 自动刷新端口间隔从 5 秒调整为 10 秒，降低空闲资源占用

## 🐛 修复

- 修复外部命令（CMD）窗口闪烁问题
- 修复配置恢复期间触发无意义的自动保存
- 修复更新资产名大小写匹配问题
