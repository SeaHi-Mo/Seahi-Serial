## v0.1.15

### ✨ 新增

- **首次使用引导** — 9 步聚光灯引导，覆盖所有核心功能
- **12 种主题** — 6 种风格（默认、浮世绘彩、诗意东方、水墨丹青、桃之夭夭、金风玉露）× 深浅色
- **WSL 端口映射面板** — USB 设备映射到 WSL，支持自动映射和手动映射
- **WSL 串口监控** — 通过 Python bridge 在 WSL 内调试串口设备
- **WSL 发行版管理** — 启动/关闭 WSL 发行版，实时显示运行时间和内存使用
- **USB 设备插拔检测** — CM_Register_Notification 监听设备变化，自动刷新列表
- **快速指令系统** — 自定义常用命令一键发送
- **终端模式** — 输出区模拟终端交互
- **提交 issue 按钮** — 一键打开 GitHub issue 页面

### 🔧 改进

- **按钮风格统一** — 未连接时低调底色，连接后主题色
- **端口下拉框缩窄** — 节省工具栏空间
- **引导卡片定位优化** — 基于实际尺寸动态计算，箭头自动跟随
- **WSL 面板窗口适配** — 支持窗口缩放和分隔线拖拽
- **时间戳精度提升** — 毫秒级时间戳
- **Toast 通知自动消失** — 错误提示 5 秒后自动关闭

### 🐛 修复

- 修复 WSL 面板发送框在窗口缩放时被挤出的问题
- 修复 WSL 拖拽分隔线计算未减去标题栏高度的问题
- 修复 quickCmd SVG 图标路径属性缺少闭合引号的问题
- 修复行号列宽被误改的问题

## v0.1.14

### ✨ 新增

- **提交 issue 快捷按钮** — 顶部栏新增"提交issue"按钮，一键打开 GitHub issue 页面
- **WSL 终端进程追踪** — 监控通过应用启动的 WSL 窗口进程 PID，关闭窗口后自动刷新状态
- **WSL 发行版运行时间秒级刷新** — 运行时间不再只在获取数据时更新，改为前端每秒自动递增
- **USB 设备 iProduct 名称显示** — 串口下拉框优先显示 USB 设备产品名称

### 🔧 改进

- **时间戳精度提升** — 时间戳从 `HH:MM:SS` 提升到 `HH:MM:SS.mmm`（毫秒级）
- **Toast 通知自动消失** — 错误横幅提示 5 秒后自动消失（仍可点击提前关闭）
- **UI 文案修正** — "自动回滚" 更正为 "自动滚动"

### 🐛 修复

- 修复关闭 WSL 窗口后应用状态不刷新的问题（增加终端进程存活检测）
- 修复行号列宽被误改为 3.5em 导致显示异常
